Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language.

Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements.

Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.

WEB DESIGN

Web design can be divided into front-end or back-end.

The Front-end part mainly involves how the websites appear and how it is interpreted by the web browsers.

The back-end mainly involves how data supplied by users can be stored into a database system and how these data can also be retrieved to provide details to users.

For front-end, web developers achieve this by using the following languages:

  • HTML
  • CSS
  • Javascript

Object Oriented Programming

Object Oriented Programming Language is a programming language that uses objects to design and develop programms and application

Each object consist of its own data and has it's own function which can manipulate the data

Examples of Object Oriented Programming Language include :

  • C++
  • C#
  • Javascript

For a programming language to be considered as an Object Programming Language, it should meet the following :

  • Support inheritance
  • Support polymorphism
  • Support Data Encapsulation
